# GitHub Pages Markdown Test

This site is generated from Markdown and deployed with GitHub Actions to GitHub Pages.

The purpose of this repo is to verify that a public Pages site can be accessed by regular users outside the organization, and that a custom domain can be added later.

## What This Proves

- Markdown content builds into static HTML.
- GitHub Actions can publish the generated site to GitHub Pages.
- The repository is ready for a future custom domain.
- Public visitors can access the site without org membership.

## Next Custom Domain Step

When you have a domain ready, add it in the repository Pages settings and create the required DNS records with your domain provider.
